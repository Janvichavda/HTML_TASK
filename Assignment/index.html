  • What are the different tags to separate sections of text?
    >< section >: This tag represents a standalone section of a document, allowing you to group related content together.
    < article >: This tag is used for self-contained compositions, such as blog posts or news articles, that can be distributed or syndicated independently.
    < div >: This tag is a generic container that can be used to group elements or separate sections of content.
    < aside >: This tag is used for content that is tangentially related to the main content, often used for sidebars or supplementary information.
    < nav >: This tag represents a section of navigation links.
    < header >: This tag represents the introductory content or a container for introductory elements at the top of a document or section.
    < footer >: This tag represents the footer of a document or section, typically containing information such as authorship, copyright, or related links.
    < main >: This tag represents the main content area of a document, excluding headers, footers, sidebars, and navigation.
    < figure > and < figcaption >: These tags are used to group media content (such as images) and provide a caption for the content.

  • What is difference between HTML and XHTML?

  • What are logical and physical tags in HTML?
    > Logical tags are used to tell the meaning of the enclosed text. The example of the logical tag is < strong > < /strong > tag. When we enclose the text in the strong tag, it tells the browser that enclosed text is more important than other texts.
    Physical tags are used to tell the browser how to display the text enclosed in the physical tag. Some of the examples of physical tags are < b >, < big >, < i >.
